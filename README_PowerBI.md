## Power BI Dashboards

### Patient Treatment Acceptance Dashboard
Built from 2,000 dental patient visits across 
5 providers and 6 insurance plans.

Visuals include:
- Monthly Production Trend (line chart)
- Total Production KPI card — $889K
- Total Patients KPI card — 2,000
- Treatment Acceptance Rate (donut chart) — 78.1%
- Decline Reasons (bar chart)
- Month slicer for interactive filtering

Data source: Excel — Patient Treatment Acceptance dataset
Tool: Power BI Desktop

Status: In progress — additional dashboards planned
as SQL portfolio database grows.
